# Q-Learning Delivery Robot | Reinforcement Learning
---
## Overview
This project demonstrates a simple implementation of **Q-Learning**, a reinforcement learning algorithm.
A robot learns to reach a goal position by choosing optimal actions over time.
---
## Key Concepts
- Reinforcement Learning
- Q-Table
- Epsilon-Greedy Strategy
- Reward Maximization
---
## How It Works
- The robot starts at position 0.
- It can move LEFT or RIGHT.
- It receives a reward only when it reaches the goal.
- Over multiple episodes, it learns the best actions.
---
## Tech Stack
- Python
- NumPy
---
## Output
- Learned Q-table
- Optimal action per state
