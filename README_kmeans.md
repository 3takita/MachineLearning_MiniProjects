# K-Means Student Clustering | Unsupervised Learning (pattern Discovery)

## Overview
This project demonstrates **K-Means clustering**, an unsupervised learning algorithm used to group similar data points.

## Problem
Group students based on:
- Sleep hours
- Coffee consumption

## Key Concepts
- Clustering
- Centroids
- Distance-based grouping

## How It Works
- The algorithm groups students into 2 clusters.
- Each cluster represents a pattern (like `high sleep` vs `high coffee`).

## Tech Stack
- Python
- NumPy
- Scikit-learn

## Output
- Cluster assignment per student
- Cluster centroids

## Areas of Application
Clustering is widely used in:
- Customer segmentation
- Recommendation systems
- Data exploration
